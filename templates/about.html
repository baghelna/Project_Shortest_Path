About Us

This is what we did.

This website was created by Nikeshsingh Baghel, Mohit Shingane and Nikhil Likhar from Computer science branch of Shri Ramdeobaba College of Engineering and Management.

We used Flask, html and CSS to build this website, for visualisation of graph we used Folium library which uses openstreetmaps. There are also libraries such as osmnx and networkx in python which we do NOT used. The reason behind not using these libraries was they have inbuilt function which directly gives the output, but we wanted to build this from scratch and in this way someone who wants to understand the concept and applications of shortest path algorithm can observe how dijkstra's algorithm finds the shortest path and displays those points in graph

Therefore wrote whole code from scratch, for calculating the shortest path we wrote dijkstra's algorithm and after getting the path we added that path to openstreetmap

Since we were not able to find any dataset, we made our own dataset to make graph for dijkstra's algorithm.
We used 20 cities of India that are -
Bangalore,Hyderabad,Kolkata,Delhi,Bhopal,Thiruvananthapuram,Lucknow ,Chennai,Srinagar,Gandhinagar,Mumbai,Ahmedabad,Pune,Nagpur,Jaipur,Chandigarh,
Guwahati,Indore,Aurangabad,Patna

If anyone can contribute for our dataset or has any queries can contact us, we'll be grateful to reply!
